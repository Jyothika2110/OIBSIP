## Description
This project is a Java console-based ATM Interface developed as part of the Oasis Infobyte (OIBSIP) internship.  
It allows users to perform basic ATM operations through a menu-driven program.
## Features
- Check account balance
- Deposit money
- Withdraw money
- Exit the application
## Technologies Used
- Java
- JDK 17+
## How to Run
1. Open Command Prompt in the project folder
2. Compile the program:
   javac ATMInterface.java
3. Run the program:
   java ATMInterface
